## **NERF TURRET**
### Arduino
Pinout -
- pan servo ---------> pin 10
- tilt servo  ---------> pin 9
- recoil servo -------> pin 11
- motor transisitor -> pin 6
- joystick (optional)
    - pan ------------> pin A7
    - tilt  ------------> pin A6
    - btn ------------> pin 3
